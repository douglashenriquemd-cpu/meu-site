# Meu Site

Um site simples feito com HTML, CSS e JavaScript.

## Como rodar

Abra o arquivo `index.html` no seu navegador. É só dar dois cliques nele!

## Estrutura

- `index.html` — estrutura da página
- `style.css` — estilos e aparência
- `script.js` — interatividade

## Tecnologias

- HTML5
- CSS3
- JavaScript
