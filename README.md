# SOS Método — Landing Page

Página de vendas (landing page) do SOS Método — curso de higienização de estofados.

## Como visualizar

Abra o arquivo `index.html` no navegador.

## Sobre

Página single-file (HTML + CSS + JS, tudo em um arquivo só) com:
- Cronômetro de oferta
- Grade de módulos do curso (accordion)
- Animações ao rolar a página
- Layout responsivo (funciona no celular)

> Nota: a grade do curso contém os módulos 1 a 11. Os módulos 12 a 19 ainda serão adicionados.
